🎯 What is RPM?

RavensOne Package Manager (RPM) is a lightweight package manager designed specifically for RavensOne modules. It's inspired by npm, cargo, and pip but optimized for simplicity and speed.

🚀 Getting Started

1. Initialize a Project

$ node scripts/rpm.js init
🎯 Initializing RavensOne project...
✅ Project initialized!

2. Install Packages

$ node scripts/rpm.js install @ravens/http-client
📦 Installing @ravens/http-client...
✅ Installed @ravens/http-client@1.0.0

3. List Packages

$ node scripts/rpm.js list
📦 Installed packages:
✓ @ravens/http-client@1.0.0

🔧 CLI Commands

# Initialize new project
raven init

# Install package
raven install @ravens/http-client

# Install as dev dependency
raven install --dev @ravens/test-utils

# Uninstall package
raven uninstall @ravens/http-client

# List installed packages
raven list

# Search for packages
raven search http

# Show package info
raven info @ravens/ui-components

# Publish package
raven publish

💻 Using Packages in Your Code

Import Syntax:

Use standard ES6-style imports to use packages in your RavensOne code. 

// Import from installed package
import { HttpClient } from '@ravens/http-client';

// Use the imported module
const client = new HttpClient('https://api.example.com');

async fn main() {
    const response = await client.get('/users');
    console.log(response);
}

main();
// Import UI components
import { Button, Card, Modal } from '@ravens/ui-components';

// Create a button
const btn = new Button({
    label: 'Click Me',
    onClick: () => console.log('Clicked!')
});

btn.mount(document.querySelector('#app'));
